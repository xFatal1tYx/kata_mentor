Что такое «анонимные классы»?

Анонимные классы — это классы, которые не имеют имени и их создание происходит в момент инициализации объекта.
Если нам нужно переопределить метод в каком-либо классе (или объявить метод в интерфейсе) можно сделать это с помощью анонимных классов:

На самом деле здесь происходит создание класса наследника, который переопределяет метод родителя:

	class Animal {
		public void eat() {}
	}
  
	Animal animal = new Animal() {
		public void eat() {
			System.out.println("Переопределение");
		};
