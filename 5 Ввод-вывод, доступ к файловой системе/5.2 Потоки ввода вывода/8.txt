Available - стоит ли использовать этот метод для проверки: есть в потоке ещё байты для чтения?

По сути метод available() возвращает количество байт доступных для чтения. 
То есть, если поток для чтения был «заблокирован», то метод вернёт 0 и это вовсе не будет означать, что конец потока был достигнут.

Метод read() блокирует поток и считывает данные, это даёт нам гарантию того, что поток будет прочитан до конца, когда метод вернёт -1.
