Перечислите все модификаторы доступа в порядке уменьшения закрытости, какую область видимости они имеют?

Модификаторы доступа — это ключевые слова, которые регулируют уровень доступа к разным частям кода. 

Всего в Java есть четыре модификатора доступа: 
private - доступ в пределах одного класса
protected - доступ в пределах пакета И для классов наследников
default (отсутствие модификатора) - доступ в пределах пакета
publuc - доступ отовсюду без ограничений

Вложенный класс/интерфейс, конструктор, метод и переменная класса может использовать любой модификатор.
Верхний класс/интерфейс может иметь только два модификатора public (только один в файле) или default.
Переменные внутри методов не имеют модификаторов доступа и являются локальными (существующими только внутри метода)
