Что делает ключевое слово static?

Если переменная или метод класса задана с модификатором static, значит её значение будет общим для всех объектов
Instance = переменная/метод объекта. 
Static = переменная/метод класса.

Обращение к такой переменной/методу происходит как имя_класса.имя_переменной (но при обращении к ней через один из объектов, ошибки не будет)

- Со static переменными/методами можно взаимодействовать ДО создания объектов (поэтому точка входа main всегда имеет static)
- Static метод/переменная НЕ МОЖЕТ взаимодействует с нестатическими 
(кроме ситуаций, когда в таком методе мы создаём объект и взаимодействуем с переменной через этот объект)
- В статических методах/переменных нельзя использовать this
- Static метод/переменную можно использовать с модификатором final
- Локальная переменная не может быть static
