Что делает ключевое слово static?

Вложенные классы/интерфейсы/enum, переменные класса и неабстрактные методы класса могут быть заданы с модификатором static, значит их значение будет общим для всех объектов.

Instance = переменная/метод/… объекта. 
Static = переменная/метод/… класса.

Обращение к статическим переменным/методам/… происходит как имя_класса.имя_переменной (но при обращении к ним через один из объектов, ошибки не будет)

- Со static переменными/методами/… можно взаимодействовать ДО создания объектов (поэтому точка входа main всегда имеет static)
- Static переменная/метод/… НЕ МОЖЕТ взаимодействовать с нестатическими (кроме ситуаций, когда мы создаём объект и взаимодействуем с ними через этот объект)
- В статических методах/переменных/… нельзя использовать this
