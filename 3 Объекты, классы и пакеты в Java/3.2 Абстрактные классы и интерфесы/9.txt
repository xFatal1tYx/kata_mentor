2.	Что такое абстрактный класс и метод?

Абстракция в Java применима к классам и методам, для этого используется слово abstract
Абстрактный класс – это класс, который содержит абстрактные методы без реализации (хотя наличие таких методов необязательно). 
Такие классы используется, когда мы не можем заранее знать логику объекта, но можем выделить его главные свойства. 

- Невозможно создать объект от абстрактного класса, это его отличие от обычного класса.
- После наследования все абстрактные методы класса должны быть определены, либо child класс также должен быть абстрактным.
- В абстрактном классе можно описать поля, которые будут отражать определённые свойства объекта или добавить обычный метод с реализацией по умолчанию.
- Абстрактные классы могут реализовывать интерфейсы и при этом не описывать все их методы.
- Абстрактный класс не может быть final.
- Абстрактный класс должен иметь конструктор (если его не задать, компилятор создаст по умолчанию). Он может пригодится для задания начальных значений общих переменных.
