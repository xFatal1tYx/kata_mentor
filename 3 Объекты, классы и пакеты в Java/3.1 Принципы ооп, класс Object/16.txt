Перечислите методы класса Enum. Расскажите, что делает каждый из этих методов?

- final String name() – возвращает задекларированное имя константы.
- String toString() – по умолчанию возвращает задекларированное имя константы, в отличие от name() может быть переопределен.
- final int ordinal() – возвращает порядковый номер определённой константы (с нуля).
- final int compareTo(E o) — позволяет сравнивать объекты enum по порядку метода ordinal().

Так же есть два статических метода, добавляемые в процессе компиляции:
- static EnumClass valueOf(String name) –  возвращает элемент перечисления EnumClass с названием, равным name. 
Если элемент не будет найден, будет выброшен IllegalArgumentException или NullPointerException если аргумент будет null.
- static EnumClass[] values()  – возвращает список всех элементов enum-класса.
Невозможно создавать экземпляры перечисления вне границ Enum, поскольку у него нет public конструктора, экземпляр перечисления создается в момент первого обращения.
