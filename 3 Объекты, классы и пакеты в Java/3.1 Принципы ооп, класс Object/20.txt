Передача параметров по ссылке или по значению? Как передаются объекты в метод?

Java всегда передает параметры по значению!

В случае с примитивами передаётся копия значения,изменения этого значения внутри метода никак не влияет на переменную.
В случае с объектами также передаётся копия, но на ссылку. 
В методе можно изменить внутренние поля объекта, к которым есть доступ, но изменить оригинальную ссылку на объект невозможно.
