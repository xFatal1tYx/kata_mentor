Как реализована неизменность String?

Прежде всего, класс String использует модификатор final, т.е. его нельзя наследовать.
Главная переменная имеет тип char[], поэтому объект String фактически инкапсулируется с этим массивом символов. 
К тому же у переменной используется модификаторы final private и отсутствует setter. 
Следовательно, значение не может быть изменено за пределами класса String, а задано лишь один раз в момент создания.
