Класс Object, все методы кроме wait, notify, notifyAll, назвать и объяснить, как работают.

Любой класс в java наследуется от класса objects (java.lang.Object). Класс object - единственный класс, у которого нет родителя.

Поэтому все типы и классы могут реализовать те методы, которые определены в классе Object:
1) Метод getClass позволяет получить тип данного объекта.
2) Метод hashCode позволяет задать некоторое числовое значение, которое будет соответствовать данному объекту.
3) Метод equals сравнивает два объекта на равенство и возвращает boolean.
4) Метод toString служит для получения представления данного объекта в виде строки.
5) Метод clone cоздает и возвращает копию объекта. Клонирование по умолчанию является поверхностным, это означает что:
- Если класс имеет только примитивные типы, то будет создана совершенно новая копия объекта и возвращена ссылка на этот объект.
- Если класс также содержит ссылочные типы, тогда ссылки на объекты этих классов скопируются. После клонирования оба объекта будут иметь одинаковые ссылки у референсных переменных.

В классе можно объявить метод finalize(), который вызывается перед сборкой мусора (уничтожением объекта). 
Но сейчас этот метод практически не используется (deprecated) из-за непредсказуемости вызова.
