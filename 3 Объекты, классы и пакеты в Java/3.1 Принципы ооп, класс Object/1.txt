Что такое конструкторы? Конструктор по-умолчанию?

Конструктор - специальный метод, который вызывается при создании объекта

new Integer(33);
public Integer(int value) {
     this.value = value;
}

В классе можно иметь несколько конструкторов с разными параметрами.
Если в классе нет конструктора, он создастся по умолчанию (без параметров и тела). Запретить это можно созданием приватного конструктора.

Для обращения из одного конструктора к другому в том же классе, используется ключевое слово this.
Если необходимо явно вызвать конструктор класса-родителя, используется слово super. Компилятор сделает это автоматически, если набор параметров совпадает.
Использование this и super в этом контексте должно быть первой строкой в конструкторе.
