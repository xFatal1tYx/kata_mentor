Что такое конструкторы? Конструктор по-умолчанию?

Конструктор - специальный метод, который вызывается при создании класса

new Integer(33);
public Integer(int value) {
     this.value = value;
}

В классе можно иметь несколько конструкторов с разными параметрами.
Если в классе нет конструктора, он создастся по умолчанию (без параметров и тела). Запретить это можно созданием приватного конструктора.
