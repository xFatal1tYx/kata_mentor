Какие есть циклы в Java? Чем отличаются друг от друга?

Циклы — это разновидность управляющих конструкций для организации многократного выполнения одного и того же участка кода.

В Java существуют 4 вида циклов:

- while — цикл с предусловием, будет повторяться до выполнения условия.
while (условие) {
     //тело цикла
}

- do while — цикл с постусловием, аналогично предыдущему, но гарантировано выполнится хотя бы 1 раз.
do {
     //тело цикла
} while (условие);

- for — цикл со счетчиком, будет повторяться пока не достигнет заданного значения;
for (int i = 0; I < 10; i++) {   // (счётчик; условие; выражение после итерации)
    //тело цикла
}

- for each — разновидность for для перебора коллекции элементов.
int[] array = { 1, 2, 3, 4, 5 };
for (var element : array) {
   //тело цикла
}
